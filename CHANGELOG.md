# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024-01-XX

### Added
- Multi-source content ingestion (YouTube, web, PDF, audio/video)
- AI-powered content generation (summaries, quizzes, flashcards)
- Content Creation Studio (articles, social posts, diagrams)
- Interactive Q&A chat with source content
- Project and category organization
- Browser extension for content capture
- Background file processing
- Manual transcript upload feature
- Content refresh functionality
- Markdown and LaTeX support in summaries
- Real-time processing status updates

### Changed
- Complete rewrite from V1 to V2
- New architecture with FastAPI backend and Next.js frontend
- Improved content extraction with multiple fallback methods
- Enhanced AI prompts for better content quality

### Fixed
- YouTube transcript extraction reliability
- PDF text extraction with OCR fallback
- Large file processing (audio/video chunking)
- Error handling and user feedback

- Workflow automation engine
